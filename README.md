# Akshitha Ganga BR — Personal Portfolio

A fully responsive personal portfolio website built with HTML5, CSS3, and JavaScript.  
🌐 **Live at:** [akshithaganga07.github.io](https://akshithaganga07.github.io)

## 👩‍💻 About

This portfolio showcases my skills, projects, certifications, and extracurricular activities as a final-year B.Tech ECE student at Ananthalakshmi Institute of Technology and Sciences, Anantapur (2023–2027).

## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Flexbox, Grid, animations, responsive design
- **JavaScript (ES6)** — DOM manipulation, scroll effects, form handling
- **Google Fonts** — DM Serif Display & DM Sans
- **GitHub Pages** — Hosting & deployment

## 🚀 Getting Started

To run locally, just clone the repo and open `index.html`:

```bash
git clone https://github.com/akshithaganga07/akshithaganga07.github.io.git
cd akshithaganga07.github.io
open index.html
```

No dependencies or build tools needed — it's pure HTML, CSS, and JavaScript.
